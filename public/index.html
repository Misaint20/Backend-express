Documentación - Express.js Server Base

Endpoints

POST /register
Registra un nuevo usuario.

Parámetros:

{
  "name": "string",
  "email": "string",
  "password": "string"
}

Respuesta:

{
  "message": "User created successfully"
}
POST /login
Inicia sesión de un usuario.

Parámetros:

{
  "email": "string",
  "password": "string"
}

Respuesta:

{
  "message": "Login successful",
  "user": {
    "id": "number",
    "name": "string",
    "email": "string"
  }
}
GET /api/users
Obtiene la lista de usuarios (sin contraseñas).

Respuesta:

[
  {
    "name": "string",
    "email": "string"
  },
  ...
]
GET /users/:id
Obtiene información de un usuario específico.

Parámetros:

id: ID del usuario (en la URL)

Respuesta:

{
  "id": "number",
  "name": "string",
  "email": "string"
}
PUT /users/:id
Actualiza información de un usuario.

Parámetros:

id: ID del usuario (en la URL)

{
  "name": "string" (opcional),
  "email": "string" (opcional),
  "password": "string" (opcional)
}

Respuesta:

{
  "message": "User updated successfully",
  "user": {
    "id": "number",
    "name": "string",
    "email": "string"
  }
}
DELETE /users/:id
Elimina un usuario.

Parámetros:

id: ID del usuario (en la URL)

Respuesta:

{
  "message": "User deleted successfully"
}
